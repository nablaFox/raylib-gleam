## Raylib on Gleam

Purely functional programming language calling the raylib C library.

1. [Install gleam](https://gleam.run/getting-started/installing/)
2. [Install raylib](https://github.com/raysan5/raylib/releases/tag/5.5)
3. Ensure you have ``erl_nif.h`` somewhere on your system (Erlang/OTP development headers)
4. Run ``./build.sh`` to create NIFs
5. Run ``gleam run``

Enjoy!
